#### 遇到的问题：
1. 从GitHub克隆代码到本地，无法显示所有分支，但是可以正常切换，切换后可以显示
2. 上传代码到GitHub，使用GitHub 提供的代码时，有时会返回一个链接地址，必须通过该链接地址访问项目，直接在个人仓库中进入无文件显示。
建议之直接使用 git push -u origin master 提交上传
